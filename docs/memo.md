# ph1
## 共通機能
### MVP
#### front
- ナビゲーションバー
    - 各画面間の遷移
- ユーザ情報入力
- セッション管理
    - ユーザ情報
#### backend
##### DB
- DDL定義（個人/Master）
    - ユーザ情報
##### API
- I/F定義
    - ユーザ情報登録
    - ユーザ情報取得

## カード検索・一覧画面
### MVP
#### front
- カード名一覧が表示
    - 画像があれば更に良し
        - 英語版で利用
- 各カードの枚数
    - 一覧画面で調整可能
    - 右下に小さめで表示
- ダイアログ
    - 表示
    - レアリティ/イラスト違い毎の枚数調整
- 絞り込み条件
    - 絞り込み機能
    - マイコレクションから戻ってきた際に絞り込み情報は保持
#### backend
##### DB（
- DDL定義（個人/Master）
    - カード情報
- データ投入
    - batch作成
##### API
- I/F定義
    - 一覧取得
        - 条件絞り込み
    - カードの詳細情報を取得
        - レアリティ
        - イラスト違い
    - ユーザに紐づく詳細枚数登録
        - レアリティ/イラスト違いも同API
    - ユーザに紐づく詳細枚数更新
    - ユーザに紐づく詳細枚数参照
- 実装
